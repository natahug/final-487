Requirements:

    Site should be responsive and have a good user experience on mobile and desktop. (10 points)
    Site must load the external data from Google API URL (10 points) https://www.googleapis.com/download/storage/v1/b/teaching-api/o/people.json?generation=1576123202877101&alt=media
    Site must display a photo slider of your choice using external data from this data file. (10 points) https://www.googleapis.com/download/storage/v1/b/teaching-api/o/photos.json?generation=1576123660992318&alt=media
    Site must have a data visualization chart (pie, bar, column, etc.) comparing something from the data like sport or gender. (20 points)
    Site should display text generated from a mathematical operation populated by a variable such as "5 of the 10 people are baseball fans" or "the group is 50% women" (20 points)
    Site should display every person in the data in an easy-to-read way that includes name, company, email, gender, sport, about, photo (20 points)
    Design is clean, usable and pleasing to users relative to a 2 hour development time (10 points)
